# workManage-WebFront
> [!NOTE]
> ### 概要
課題管理システムのwebフロントエンドバックエンドはgasを採用

> [!NOTE]
> ### 機能 
- PWAによるWEBアプリ化
- キャッシュ機能によるオフライン時表示(
データ更新はオンライン)
- 完了・未完了の個別表示・切り替え
  
> [!NOTE]
> ### バージョン
<ユーザー>    利用者の体感に影響を与える内容  
<サイレント>  内部のリファクタリングや改善  
- ### v1.5
  - <ユーザー>各課題の残り時間を計算し24時間未満なら赤文字＋分単位で表記 24時間以上は日単位で表示するように調整
  - <サイレント>CSSを整理
